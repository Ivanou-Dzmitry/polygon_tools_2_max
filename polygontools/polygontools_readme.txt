Instalation
It is better to remove the old version of PT (1.4 and etc.). Delete "polygontools" directory in "\ENU\scripts", loader in "ENU\scripts\startup" and linked files "\ENU\usermacros" (POLYGON-PolygonXXX.mcr)

First of all move icons named "polygontools_XXX.bmp", where XXX 16i/a or 24i/a
from
\ENU\scripts\polygontools\pt_modules\icons
to
\ENU\usericons

First time After Install PolygonTools 2:
1. Run 3ds Max.
2. Open file pt_loader.ms (Scripting -> Open Script)
3. Tools -> Evaluate All (close MAXScript window after this action)
4. Customize -> Customize User Interface and go to "Toolbars" Tab
5. Find Category "POLYGON"
6. Select icon "PolygonTools 2" and move to an empty place at the Main Toolbar - button will be created.
7. Run it and Enjoy!

After Update PolygonTools 2:
1. Run Installer.
2. Run 3ds Max and Enjoy Updated Version!
